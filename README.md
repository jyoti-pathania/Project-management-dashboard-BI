## 📌 Project Overview

A multi-page, interactive Power BI dashboard built for a UK telecom infrastructure company to track and analyse **project financial performance**, **client revenue**, and **forecast accuracy** across a 6-year period (2020–2026).

The dashboard provides senior stakeholders with a single source of truth for revenue, accruals, spend, and profitability — broken down by client, month, and project stage.

---

## 🎯 Key Business Insights Delivered

- 📦 **£10.50M total revenue** tracked across **7 active clients** with a **21% profit margin**
- 💰 **£3.04M accrued (uncollected) revenue** identified — flagged for collection follow-up
- 📉 **2026 sales tracking 11% achievement** against a £16M full-year forecast - early warning for management
- 🔁 **Client1 showed 132% MoM growth** (Feb - Mar 2026); Client6 dropped 77% - client risk visibility
- 📊 **Forecast vs Actual gap** visualised year-on-year (2024–2026): consistently below forecast, prompting strategy review
- 💸 **£1 spend returns £1.26 revenue** - efficiency metric surfaced in KPI cards
- 📋 **95% payment collection rate** across all months in the report view

## 📂 Dashboard Pages (7 Tabs)

| Tab | What It Shows |
|-----|--------------|
| 📊 Accrued Overview | KPI cards, accrued revenue by project type, revenue vs provision by client |
| 🏆 Client Performance | Month-on-month sale comparison table, net sale by month chart |
| 🌍 Yearly Overview | Year-on-year revenue by client, forecast vs actual by year |
| 📅 Monthly Performance | Revenue vs forecast monthly trend, profit/loss by month |
| 💰 Revenue Trends | 6-year revenue growth story (2020-2026): revenue, accruals, provision, cost |
| 📋 Report View | Detailed financial table: sales, expenses, profit, collection rate, outstanding |
| 🎯 2026 Forecast vs Achv | Client-level forecast tracking with achievement % and visual trend |

## 🛠️ Tools & Techniques Used

- **Power BI Desktop** - report design and publishing
- **DAX** — KPI measures, MoM % change, YTD calculations, forecast achievement %
- **Power Query (M)** - data transformation and cleaning
- **Data Modelling** - relationships across client, project, financial tables
- **Conditional Formatting** - profit/loss colour coding (green/red), threshold alerts
- **Slicers & Filters** - client selector, year filter, clear filters button
- **Custom Visuals** - multi-tab navigation, grouped bar charts, line/bar combo charts
